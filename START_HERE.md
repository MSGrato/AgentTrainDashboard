# START HERE

If you are a student, do this first.

## Step 1
Run:

```bash
npm install
npm run dev
```

## Step 2
Open this file:

- `src/data/projectData.ts`

That is the main file you should edit.

## Step 3
Put your screenshots, diagrams, slides, and reports here:

- `public/project-assets/`

## Step 4
Reference those files in `src/data/projectData.ts` like this:

- `/project-assets/my-screenshot.png`
- `/project-assets/final-slides.pdf`

## Step 5
Replace all text in brackets, for example:

- `[Project Title]`
- `[Student Name]`
- `[Feature or Flow Name]`

## Step 6
When you are done, test:

```bash
npm run build
```

## Important

You usually do NOT need to edit:

- `src/App.tsx`
- `src/styles.css`

Only touch those if you want to redesign the page itself.
